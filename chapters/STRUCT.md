# Tree structure of the directory

To make the script work the directory must be structured in this way:

```md
chapters
├─── 001
│    ├─── 001.jpg
│    ├─── ...
│    └─── n.jpg
├─── 002
│    ├─── 001.jpg
│    ├─── ...
│    └─── n.jpg
├─── ...
└─── n
     ├─── 001.jpg
     ├─── ...
     └─── n.jpg
```
